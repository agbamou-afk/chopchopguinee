# CHOP Maps — Guide Opérateur Terrain

Public : agents terrain, capitaines de zone, admin pilote.
Langue : français-first, exemples courts.

## 1. Soumettre un lieu (place)
1. Ouvrir l'app sur place, autoriser la géolocalisation.
2. Vérifier que le pointeur GPS est stable (attendre 5–10 s).
3. Saisir le nom usuel ("Pharmacie Kaloum"), pas le nom légal.
4. Ajouter un repère ("en face de la station Total").
5. Choisir la catégorie correcte.
6. Soumettre → statut **"À confirmer"**.

## 2. GPS refusé / signal faible
- Si le téléphone refuse la localisation : ouvrir réglages, autoriser CHOP, puis recharger.
- Si le signal est faible : marcher 10 m vers une zone dégagée avant de soumettre.
- Ne jamais inventer des coordonnées. Mieux vaut sauvegarder un brouillon.

## 3. Brouillon hors-ligne
- Si la connexion coupe pendant la soumission, l'app sauvegarde automatiquement un **brouillon**.
- Aller dans la page Visites → onglet Brouillons.
- Quand la connexion revient : cliquer **Renvoyer**.
- Si le brouillon échoue plusieurs fois : cliquer **Supprimer** et resoumettre manuellement.
- ⚠️ Les photos ne sont pas mises en file d'attente. À prendre quand la connexion est revenue.

## 4. Décrire un repère clairement
Bon : "Carrefour Bellevue, juste après le pont, côté droit, mur jaune."
Mauvais : "Près du marché." (trop vague)

## 5. Éviter les doublons
- Avant de créer un lieu, chercher d'abord son nom dans la barre.
- Si un lieu existant est à moins de 50 m : ne pas en créer un nouveau, signaler plutôt.
- Les doublons détectés tombent dans la file d'attente admin.

## 6. Signaler un mauvais lieu
- Coordonnées fausses, nom trompeur, lieu fermé : utiliser **Signaler** sur la fiche.
- Ajouter une note ("fermé depuis 2 mois").

## 7. Vocabulaire des statuts
- **À confirmer** : soumis, en attente de vérification admin. Ne pas l'utiliser comme adresse officielle.
- **admin_verified** : vérifié par un admin. Sûr à utiliser.
- **trusted** : vérifié + corroboré par plusieurs sources. Utilisé par défaut.

## 8. Quand escalader
- Doute sur la zone/quartier → capitaine de zone.
- Conflit de coordonnées entre deux agents → admin pilote.
- Bug app (carte blanche, plantage) → capitaine + capture d'écran.
